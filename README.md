
## todo app 

how to run in local computer

```bash
npm i 
# next
npm run build
# next
npm start

```

### Feature:
- Users can create a task by just entering the title and description.
- Users can delete tasks.
- Users can edit tasks.
- users data will stored in Local Storage

### Technology:
- NextJs
- Redux
- TypeScript
- TailwindCSS
- sweetalert2
- shadcn ui
- HTML CSS
